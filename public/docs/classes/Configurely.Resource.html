\Configurely

\ConfigurelyResource

Provides a base model for resourceable types

This class implements the following patterns:

  1. STI Pattern http://www.colorfultyping.com/single-table-inheritance-in-laravel-4/ The purpose of this pattern is to provide all models with single-table-inheritance so multiple models of the same base data type can be stored in a single table.
  2. Validation Pattern http://daylerees.com/trick-validation-within-models The purpose of this pattern is to provide model-specific validation rules.
